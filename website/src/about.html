History

Team

We started the Ulight project to allow the Front-End Developers to build websites with more original designs and layout. Our modo is to make the developer's life easier by providing them a light and efficient Framework to work with. We have realized that, with stardardization, the websites looked often the same nowadays. But as developers, it is hard to focus both on an original layout and an efficient code. So we started to work on Ulight!
